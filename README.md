# HUAWEI KLVC-WXX9 黑苹果EFI配置 i5-10210U UHD620

### 适用机型
     HUAWEI KLVC-WXX9 2020（i7版本也可以用）
    
     

![nootbook](./nootbook.png)

###已驱动
- 核显
- 声卡
- USB（Type-C）
- 电池电量显示
- 键盘
- 触控板
- 亮度调节
- 蓝牙
- 变频
- WIFI

### 适用的 macOS  版本
- Big Sur
- Catalina 
- Ventura
- Sonoma
- Sequoia
- Monterey
